# Vulkan glTF demo

View glTF model in Vulkan,.

Currently the source code is based on [Vulkan Tutorial](https://vulkan-tutorial.com/).

## Dependencies

* [GLFW](https://www.glfw.org/download.html)
* [GLM](https://glm.g-truc.net/0.9.9/index.html)
